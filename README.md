# Seja bem-vindo ao School-App (Light Edition) 🚀
Sistema de cadastro de professor e estudante focado em consultas simples e complexas.
## Funcionalidade
- Cadastrar e gerenciar **estudante**👨‍🎓;
- Cadastrar e gerenciar **professor**👨‍🏫;
- Criação de **turma** baseado no gerenciamento do **professor**🎒;
- Consultas simples mostrando todos os **alunos** e **professores**🔍;
- Consultas complexa através dos atributos específico de cada **professor** e **aluno**🕵️.
